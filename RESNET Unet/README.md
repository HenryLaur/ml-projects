# Resnet UNET

Unet with resnet connections. This is better than simple unet but not as good as attention unet. It does not converge as quick.

### Losses
| Type      | Epoch | Loss   |
|-----------|-------|--------|
| Attention | 14    | 0.0079 |
| Resnet    | 34    | 0.0245 |
| Regular   | 30    | 0.0316 |
